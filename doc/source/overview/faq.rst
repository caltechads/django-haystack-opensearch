^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

See the :doc:`installation` guide for detailed installation instructions. The recommended methods are:

- Using ``uv tool``: ``uv tool install django_haystack_opensearch``
- Using ``pipx``: ``pipx install django_haystack_opensearch``
- Using ``pip``: ``pip install django_haystack_opensearch``
- From source: Clone the repository and run ``uv sync``

How do I report a bug?
^^^^^^^^^^^^^^^^^^^^^^

When reporting a bug, please include:

1. **Error message**: The complete error output, including the traceback
2. **Environment**: OS, Python version, django_haystack_opensearch version
3. **Steps to reproduce**: The exact steps to reproduce the bug

Example bug report:

.. code-block:: text

    Error: Feature 1 error

    Traceback:

    ````
    [Include traceback here]
    ````

    Steps to reproduce:

    1. Step 1
    2. Step 2
    3. Step 3
