### Installation

```bash
# Install
pip install django_haystack_opensearch

# With uv
uv tool install -p 3.13 django_haystack_opensearch
```

### Usage

```bash
# Show version info
django_haystack_opensearch version

# Fix formatting issues in an RST file
django_haystack_opensearch settings

# Show help
django_haystack_opensearch --help
```
